# Markdown

## 标题
> # 一级标题
> ## 二级标题
> ### 三级标题
> #### 四级标题
> ##### 五级标题
> ###### 六级标题
> ####### 不能更多了。
## 列表

> #### 方法一

- 3

## 代码框
使用\`来标记行代码`code`，注意用输入法  
使用 ``` 来标记块代码
```  
    a=range(10)
    for i in a
        print(i)
```
## 分割线
分割线语法是三个`*`号，例如：  
***  
## 表格  
| Tables        |Are            |Cool   |  
| :-            | :-:           | -:    |
|col 3 is       |right-aligned  |$1600  |
|col 2 is       |centered       |$12    |
|col 1 is       |left-aligned   |$5     |  
|zebra stripes  |are neat       |$1     |  
请注意`:-`表示左对齐，`:-:`表示中间对齐，`-:`表示右对齐
